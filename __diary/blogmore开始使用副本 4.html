blogmore

一个小型的博客构建工具。

安装

$ [sudo] npm install -g blogmore

初始化

$ bm init

新建博客

$ bm new -t [博客名称]
$ bm new --title [博客名称]

在 source 文件夹下根据博客初始模板新建一篇博客。

构建博客

$ bm build

本地预览

$ bm server

发布博客

$ bm publish
